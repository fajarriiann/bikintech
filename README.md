## Installation

```sh
$ git clone https://github.com/fajarriiann/bikintech.git
$ cd bikintech
$ composer install
$ npm install
$ chance .env.example to .env
$ chance setting to database
$ generate key : php artisan key:generate
$ call action to Migrate Database & Seeder : php artisan migrate
$ start server : php artisan serve
$ start vite : npm run dev
```
